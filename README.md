# 🌸 Portafolio de Karen Valdes

Este es mi portafolio profesional, desarrollado como **Trabajo Práctico Final** del **Módulo 1 del curso de Frontend** en **ADA ITW**.

## 📄 Contenido del sitio

- **Perfil profesional**: orientación laboral y formación actual
- **Formación académica**: estudios y cursos complementarios
- **Proyectos destacados**: trabajos realizados durante el curso
- **Cita inspiradora**: frase de Ada Lovelace (en honor al nombre de la fundación)
- **Formulario de contacto**: para consultas y mensajes
- **Footer**: enlaces de navegación y contacto

## 🛠️ Tecnologías utilizadas

- **HTML5 semántico** (uso de `<section>`, `<nav>`, `<footer>`, `<blockquote>`)
- **CSS3** (Flexbox, Transiciones, Posicionamiento `sticky` y `fixed`)
- **Diseño responsivo** (Media Queries para tablets y móviles)
